# Image Background Remover 🌀

AI-powered image background removal tool.

## Features

- Remove image backgrounds automatically
- Support batch processing
- High-quality output with transparent backgrounds

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py --input input.jpg --output output.png
```
